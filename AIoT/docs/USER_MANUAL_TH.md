# คู่มือการใช้งาน (User Manual)

## บทนำ

คู่มือสำหรับผู้เข้าร่วมกิจกรรมเพื่อใช้งานแอปพลิเคชัน AIoT Activity Hub

## ขั้นตอนใช้งาน

1. การลงชื่อเข้าใช้
- เข้าสู่ระบบผ่าน URL ที่ผู้จัดกิจกรรมกำหนดให้

2. ข้อมูลการล็อกอิน
- กรอก Team ID และรหัสผ่านที่ได้รับจากทีมงาน

3. การเชื่อมต่ออุปกรณ์ IoT
- ไปที่เมนู Devices ทางแถบด้านซ้าย

4. การเพิ่มอุปกรณ์ Edge
- คลิก + Add Device
- กรอกหมายเลข MAC Address ของบอร์ด Raspberry Pi Pico W หรืออุปกรณ์ที่ใช้
- ระบบจะสร้าง MQTT Topic เฉพาะสำหรับทีมโดยอัตโนมัติ

5. การจัดการ Dashboard
- ไปที่เมนู Dashboard

6. การสร้างวิดเจ็ต
- คลิก Add Widget
- เลือกรูปแบบที่ต้องการ เช่น Line Chart สำหรับอุณหภูมิ หรือ Gauge สำหรับความชื้น

7. การผูกข้อมูล
- ผูก Widget กับ MQTT Topic ที่ต้องการให้แสดงผล

8. การเข้าสู่เมนู AI Trigger
- ไปที่เมนู Automation (AI Rules)

9. การตั้งค่าเงื่อนไข Automation
- ตัวอย่าง: ถ้า [อุณหภูมิ > 30°C] และ [AI ตรวจพบใบหน้าคน] ให้ [ส่งคำสั่ง ON ไปที่ Topic /relay/1]
